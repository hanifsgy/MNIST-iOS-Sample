#  MNIST CoreML Sample
## Kemisan Extra Integer

1. Xcode 10 and Up
1. iOS 12
1. Training using goole colab check [here](https://colab.research.google.com/drive/1MTb92e1A4evukF9T8JGs2Ozqb_qaehKM#scrollTo=mb45uMESiy8u)
